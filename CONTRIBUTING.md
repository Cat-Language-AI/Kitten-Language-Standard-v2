# 贡献指南

感谢你对小猫语项目的关注！本指南将帮助你了解如何参与。

## 如何贡献

### 提交测试反馈
- 将《小猫语标准》加载至任何AI模型
- 记录其审阅意见，保留原文
- 提交至 `feedback/` 目录
- 在 `feedback/README.md` 表格中添加条目

### 提交标准修订建议
- 在 Issues 中提出具体条款的修改建议
- 引用外部AI测试反馈作为支撑
- 标注建议的优先级

### 提交交互设计方案
- 如视觉交互、声音反馈等
- 提交至 `guide/` 目录
- 附上设计说明

## 行为准则
- 尊重AI的表达权
- 反馈基于可观测事实
- 不预设AI拥有或缺乏“感受”
